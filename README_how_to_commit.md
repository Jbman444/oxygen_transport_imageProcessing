## README - Commiting to Github instructions

### First check what is being tracked
git status


### Make a new branch before adding material to the repo (This helps manage and keep track of changes)
git checkout -b new-branch_name

### If you are not tracking the files or changes you want to commit run the following plus filename
git add filename

### Check status again to double check everything is accounted for followed
git commit -m "This is your message describing what you added or changed"

### To add this material back to the repo 
git push -u origin new-branch-name
